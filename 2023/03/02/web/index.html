基于springboot的web小demo

web-demo

简单介绍下这个小demo

  • demo效果简介：

    响应浏览器的 /user 请求 ，返回数据库中的数据。

  • 环境说明：数字表示版本号

    • springboot : 2.7.9
    • mybatis : 2.3.0
    • druid : 1.2.8 (阿里的数据源)
    • mysql-connector: 8.0.19
    • lombok: 1.18.26

  • 效果演示：

    • idea 启动 WebDemoApplication 如图 服务在本地的8080端口启动
    • 使用浏览器访问localhost:8080/user/1 如图显示页面响应结果

  • 总结：

    • 使用了durid数据源的话，需要额外引入jdbc的依赖，不然会连不上数据库。

    • 写配置文件的时候引入配置处理器configuration-processor 方便写相应的配置，会有提示。

    • 使用mybatis时需要启动类上配置包扫描才能找到调用的mapper ，使用MapperScan(“mapper文件所在的包名”)，另外还需properties文件中配置mapper对应xml文件的存放位置 : mybatis.mapper-locations=classpath:mapper/*.xml

    • 详细代码可见：https://github.com/xilou37/web-demo.git
